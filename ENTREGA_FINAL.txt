┌─────────────────────────────────────────────────────────────────────────────────┐
│ 🔍 TROUBLESHOOTING                                                              │
└─────────────────────────────────────────────────────────────────────────────────┘

Gateway offline en frontend?
  → docker-compose ps
  → docker-compose logs -f gateway

Puerto 80 en uso?
  → docker-compose down
  → En Windows: netstat -ano | findstr :80

Maven descarga lenta?
  → Es normal la primera vez
  → Cache se guarda en ~/.m2/repository

¿Necesito Node.js?
  → Sí, para npm run dev
  → NodeJS >= 18 recomendado

¿Apache NetBeans?
  → Abre proyecto en backend/
  → O simplemente usa IntelliJ IDEA con Maven
